Comment ajouter un produit ?

Open Food Facts est un projet collaboratif et citoyen auquel chacun peut contribuer en ajoutant des produits ou en complètant les informations sur les produits.

Nous essayons de proposer l'interface la plus conviviale possible pour qu'il soit aisé et rapide d'ajouter des produits. Voici en images les étapes de l'ajout d'un produit, avec quelques astuces pour aller plus vite.

Avant toute chose, s'inscrire sur Open Food Facts puis s'identifier

Si ce n'est pas encore fait, vous pouvez vous inscrire en 30 secondes.

Prendre des photos du produit

Le plus simple est de prendre les photos dans l'ordre où vous en aurez ensuite besoin :

  1. le code barre
  2. le produit de face : le mieux est de le poser sur le dos sur une surface blanche (une feuille de papier par exemple) et de prendre la photo à la verticale
  3. la liste des ingrédients
  4. la composition nutritionnelle

Vous pouvez prendre les photos en mode portrait ou paysage, le cadrage n'a pas d'importance (vous pourrez les recadrer directement sur le site), mais il faut que les photos soient bien droites.

Bien sûr plus la photo est nette, moins il y a d'ombres et de reflets, mieux c'est, mais cela sera sûrement le thème d'un autre tutoriel !

A noter : il n'est pas nécessaire de "retravailler" les photos dans un logiciel avant de les envoyer sur le site (redimensionnement, recadrage, balance des blancs etc.). Tout celà sera fait directement sur le site.

Ajouter le produit

Sur le site, dans la colonne de droite, vous trouverez un formulaire "Ajouter un produit".

Le mieux est d'envoyer la photo qui contient le code barre en cliquant sur le bouton "Envoyer une image".

A noter que si vous utilisez le navigateur Chrome ou Firefox, la photo est automatiquement redimensionnée à une taille raisonnable afin que son envoi soit très rapide même si vous avez un appareil photo avec 20 millions de pixels.

Si la photo du code barre n'est pas trop floue, il sera reconnu automatiquement. Si le produit est déjà dans la base, vous serez redirigé vers la page du produit et vous pourrez vérifier que toutes les informations sont présentes. Si vous pensez avoir de meilleures photos, n'hésitez pas à modifier la fiche pour les envoyer. Si le produit n'existe pas encore dans la base, vous obtiendrez directement le formulaire d'ajout de produit.

Envoyer la photo de face du produit

A moins que la photo qui contient le code barre soit aussi la photo de face du produit, cliquez sur "Ajouter une photo" pour l'ajouter.

Vous pouvez redresser l'image en cliquant sur les boutons "Pivoter".

Sélectionnez ensuite la zone de l'image qui correspond au produit en faisant un "cliquer/glisser" avec la souris, puis cliquez sur le bouton "Enregistrer et/ou recadrer l'image".

Si vous avez pris la photo sur un fond blanc, vous pouvez essayer de cocher la case "Photo sur fond blanc : essayer d'enlever le fond" ce qui permet de rendre blanc les bords légèrement gris de la photo et de corriger et renforcer les couleurs :

Ajouter les informations sur le produit

Le nom du produit est celui qui figure sur l'emballage.

La dénomination générique est le nom "légal" du produit, sans indication de marque. Il figure la plupart du temps sur le produit.

Les champs "Conditionnement", "Marques", "Catégories", "Origines", "Labels" sont des listes de mots-clés (tags) sous lesquelles le produit sera référencé. Vous pouvez taper ces mots-clés en les séparant par une virgule ou la touche entrée.

Vous pouvez entrer tous les mots-clés qui vous paraissent pertinents (en vous inspirant des produits déjà référencés par exemple). Lorsque le nombre de produits dans la base sera plus significatif, on pourra faire le tri parmi tous les mots-clés et proposer une arborescence de catégories, des correspondances entre plusieurs orthographes ("Fromage" ou "Fromages") etc. On pourra également proposer des suggestions à la volée (en ne tapant que quelques lettres) pour les marques etc.

Envoyer la photo des ingrédients

De la même manière que la photo de face du produit, envoyez la photo de la liste des ingrédients.

Sélectionnez la zone de l'image qui contient la liste, puis cliquer sur "Enregistrer et/ou recadrer".

Pour éviter de devoir recopier une longue liste d'ingrédients, vous pouvez essayer le bouton "Extraire le texte des ingrédients". La reconnaissance des ingrédients marche d'autant mieux que la photo est droite et nette. D'autre part si le texte est courbé (photo d'une canette par exemple) ou si le fond n'est pas uniforme, les ingrédients ne sont généralement pas reconnus.

Avec ce produit, la reconnaissance est quasi-parfaite (un : a été reconnu comme un 1). C'est malheureusement rarement le cas, et il faut donc souvent corriger le texte extrait automatiquement.

Cette reconnaissance automatique va s'améliorer avec le temps : avec un grand nombre de produits référencés, on peut construire un dictionnaire des ingrédients qui est une aide précieuse pour l'algorithme de reconnaissance textuelle.

Indiquez ensuite les "traces éventuelles" de produits potentiellement allergènes. Ces indications sont très utiles, et parfois vitales, pour les personnes allergiques.

Envoyer la photo des informations nutritionnelles

De plus en plus de produits indiquent la composition nutritionnelle sur l'emballage. Comme pour les ingrédients, envoyez et recadrez la photo correspondante.

Le remplissage du tableau n'est pas encore automatique (les tableaux sont plus difficiles à reconnaître que la liste des ingrédients), mais celà viendra peut-être.

En attendant le tableau est pré-rempli avec les nutriments les plus souvent listés.

La législation européenne autorise les fabricants à lister d'autres nutriments (vitamines et minéraux en particulier). S'ils sont listés sur l'emballage, vous pouvez les ajouter en tapant leur nom dans la dernière ligne du tableau. En tapant les premières lettres vous obtiendrez une liste déroulante avec les nutriments correspondants et les unités correspondantes les plus communes (mg pour la vitamine C par exemple). Si vous ne trouvez pas le nutriment, vous pouvez le taper en toutes lettres.

Enregistrer le produit

Il ne vous reste plus qu'à cliquer sur le bouton "Enregistrer".

Merci d'avance pour tous les produits que vous ajouterez dans la base !
