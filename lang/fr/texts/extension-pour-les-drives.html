Extension Open Food Facts pour les courses en ligne

Faites des choix plus informés lors de vos courses en ligne

DriveOFF est une extension (open source) pour les navigateurs internet Chrome et Firefox vous permettant de mieux consommer en affichant en temps réel les informations nutritionnelles des produits que vous consultez sur de nombreux sites d'achat alimentaire en ligne.
Le bandeau DriveOFF affiche les informations suivantes pour un produit donné:
  • Nutri-Score (qualité nutritionnelle d’un produit)
  • Groupe Nova (transformation d’un produit)
  • Eco-Score (impact environnemental d'un produit)
Les informations sont affichées sur les listes de produits (pour vous permettre de les comparer facilement entre eux), et sur le produit lui même.

Les données

DriveOFF ne pourrait exister sans le projet libre Open Food Facts. Vous pouvez contribuer via les apps Android et iOS et visiter la version web Open Food Facts

Installer DriveOFF

DriveOFF est pour désormais disponible sur le Chrome WebStore et le Magasin d'extensions Mozilla pour Firefox.
