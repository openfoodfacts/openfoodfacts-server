La inteligencia detrás de las aplicaciones y API de Open Food Facts

Open Food Facts recopila datos utilizando una API WRITE, los procesa y los pone a disposición utilizando una API READ y una versión web. Calcular el Nutri-Score, los grupos NOVA, la extracción de aditivos, independientemente de su idioma, sinónimos o posibles errores ortográficos. Esto es solo una pequeña fracción de los temas cubiertos por el equipo del servidor, para garantizar la transparencia de los alimentos a escala planetaria.

Contribuye al núcleo usando Perl

¿Quieres aprender un nuevo idioma? El servidor está principalmente codificado en Perl. Ya sea que seas un experimentado adicto a Perl, o que hayas tocado Perl para las tareas de sys-administration, hay un error para ti, y mentores que te guiarán. Y también más de una forma de hacerlo ;-)

Crea herramientas útiles en cualquier idioma

HungerGames es el ejemplo de un pequeño juego codificado en react que está teniendo un impacto positivo en Open Food Facts. LanguageSwitcher, otra herramienta, está codificada en Python. Tu herramienta, tu idioma (por favor, evita la ensambladura ;-)
