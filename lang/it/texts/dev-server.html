L'intelligenza dietro le app e le API di Open Food Facts

Open Food Facts raccoglie dati usando un API WRITE, li elabora e li rende disponibili usando un API READ ed una versione web. Calcolando il Nutri-Score, i gruppi NOVA, estraendo additivi, indipendentemente dalla loro lingua, sinonimi o potenziali errori di ortografia. Questa è solo una piccola frazione degli argomenti coperti dal team server, per assicurare la trasparenza alimentare su scala planetaria.

Contribuisci al Core usando Perl

Vuoi imparare una nuova lingua? Il server è principalmente programmato in Perl. Che tu sia un esperto di Perl, o che tu abbia toccato Perl solo per mansioni di amministrazioni di sistema, c'è un bug della tua taglia e mentori che ti guidano. E anche più di un modo per farlo ;-)

Crea strumenti utili in qualsiasi lingua

HungerGames è l'esempio di un piccolo gioco programmato in react che sta avendo un impatto positivo su Open Food Facts. LanguageSwitcher, un altro strumento, programmato in Python. Il tuo strumento, la tua lingua (per favore, evita l'Assembler ;-)
