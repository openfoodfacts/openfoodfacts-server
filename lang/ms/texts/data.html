Maklumat mengenai medan-medan berbeza untuk longgok MongoDB dan eksport CSV telah tersedia di https://world.openfoodfacts.org/data/data-fields.txt

Eksport Delta

Eksport delta harian disediakan selama 14 hari sebelumnya. Senarai fail delta semasa yang tersedia boleh diperolehi di https://static.openfoodfacts.org/data/delta/index.txt . Setiap baris mewakili satu fail yang tersedia di https://static.openfoodfacts.org/data/delta/{filename}. Nama fail mengandungi setem masa UNIX bagi perubahan pertama dan terakhir yang terkandung di dalam fail JSON, supaya fai-fail delta boleh diimport (setelah pengekstrakan) dengan mongoimport dalam tertib berabjad.

Harap maklum, disebabkan sifat mongoexport, fail-fail delta tidak dapat memberitahu anda berkenaan produk-produk yang terpadam. Untuk membuang produk-produk terpadam dari pangkalan data anda, anda perlu mengimport longgok MongoDB sepenuhnya.

Eksport Data CSV

Data untuk semua produk, atau sebahagian produk, boleh dimuat turun dalam format CSV (boleh-baca menerusi OpenOffice, Excel dan lain-lain perisian) melalui borang gelintar lanjutan.

Pautan
https://static.openfoodfacts.org/data/en.openfoodfacts.org.products.csv

Pengekodan fail ialah Unicode UTF-8. Aksara yang memisahkan medan-medan ialah <tab> (penjadualan).

API JSON Langsung

Satu API JSON juga tersedia untuk membaca data sesebuah produk. API ini khususnya digunakan dalam aplikasi mudah alih Open Food Facts untuk iPhone dan Android.

Kami telah mengubah API dari masa-ke-semasa tetapi kami berhati-hati bagi keserasian yang terdahulu. Sila beritahu kami jika anda telah menggunakannya, supaya kami dapat memberitahu anda berkenaan perubahan-perubahan yang ada.

URL to read data for a product: https://world.openfoodfacts.org/api/v0/product/[barcode].json

Contoh: https://world.openfoodfacts.org/api/v0/product/737628064502.json

Dokumentasi lanjutan tersedia dalam Wiki . (Perhatian penting: halaman telah dibahagikan kepada beberapa sub-halaman untuk BACA dan TULIS). Satu dokumentasi yang lebih baik sedang dibangunkan, dengan menggunakan kes-kes untuk memaparkan satu produk, dan membolehkan pengguna anda menambahkan data yang diperlukan untuk mendapatkan Nutri-Score, kumpulan NOVA dan bahan tambahan. Anda boleh mencapainya di sini .

Anda sangat dialu-alukan menggunakan API untuk tujuan-tujuan pengeluaran, asalkan 1 panggilan API = 1 imbasan sebenar oleh seorang pengguna. Sebarang percubaan untuk mengikis pangkalan data menggunakan API berkemungkinan akan disekat, kerana eksport harian penuh tersedia dalam halaman ini.

Dan juga, penghantaran satu tajuk HTTP dengan panggilan API anda adalah cara yang baik untuk kita berhubung, jika ada masalah.

Sekiranya anda tidak pasti berkenaan apa-apa yang ada di sini, atau mempunyai sebarang pertanyaan (bukan pertanyaan bodoh), silalah bertanya di Slack :-)

Apl Android dan iPhone

Kod untuk apl mudah alih Open Food Facts telah tersedia di GitHub ( Android dan iOS ). Apl ini membolehkan para pengguna mengimbas barkod produk, melihat maklumat produk, dan mengambil serta menghantar gambar dan data bagi produk yang tiada.

Apl untuk iOS dibangunkan dengan Swift.

Aplikasi untuk Android dibangunkan dengan Java.

Anda dialu-alukan untuk mengguna-semula kod (ia sumber terbuka) dan bantu kami menambahbaiknya, untuk semua orang, di planet ini.
