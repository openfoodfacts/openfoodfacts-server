Una base de datos de productos cosméticos

Open Beauty Facts es una base de datos de productos cosméticos con ingredientes, alérgenos, información y todos los fragmentos de información que podemos encontrar en las etiquetas de los productos.

Hecho por todos

Open Beauty Facts es una asociación de voluntarios sin fines de lucro.

Más de 1800 colaboradores como vos han agregado más de 9 000 productos de 150 países usando nuestra aplicación para Android, iPhone o Windows Phone, o su cámara para escanear códigos de barras y subir imágenes de los productos y sus respectivas etiquetas.

Para todos

Los datos sobre cosméticos son de interés público y deben ser abiertos. Toda la base de datos se publica como datos abiertos y puede ser reutilizada por cualquier persona y para cualquier uso. Echa un vistazo a las geniales reutilizaciones, ¡o haz la tuya!

Hacer mejores elecciones de cosméticos

El conocimiento es poder. ¡El conocimiento de los cosméticos es poder para mimarte mejor!

  • Descodificar etiquetas

    Aditivos cosméticos, códigos FIL, números CI, alérgenos, códigos de embalaje... Open Beauty Facts te ayuda a darle sentido a la letra pequeña en la etiqueta de los productos.

    También valoramos los riesgos de cada producto.

  • Encontrar productos

    Nuestro formulario de búsqueda incluye docenas de criterios para que pueda encontrar un producto que coincida con sus criterios.

  • Comparar y cambiar

    ¿Sabe qué champú tiene menos productos químicos? En 3 clics puede crear un gráfico interactivo para mostrarlo. Por lo tanto, puedes cambiar por champús más saludables y al comparar productos de diferentes marcas, fomentar a los productores a mejorarlos.

  • Explorar y descubrir

    ¿Tienes curiosidad por la cosmética? Explore productos a partir de sus facetas, como marcas, categorías, etiquetas, orígenes de ingredientes, etc. Las facetas se pueden combinar para descubrir nuevas ideas, por ejemplo, para mapear los orígenes de los ingredientes que llevan una etiqueta de comercio justo.

Cosméticos, para la innovación y la ciencia

¡Nuestros datos se pueden reutilizar para el ámbito educativo, la elaboración de nuevos productos y servicios, la investigación y más!

La base de datos Open Beauty Facts es una base de datos abiertos publicados bajo la licencia Open Database. Cualquiera puede usarlo para cualquier propósito, comercial o no comercial, siempre que se atribuya la fuente de los datos y que las mejoras a la base de datos se compartan de manera similar.
