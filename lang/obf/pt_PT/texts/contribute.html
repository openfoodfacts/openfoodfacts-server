Todos podem contribuir

Faça parte da nossa base de dados colaborativo, gratuito e aberto de produtos cosméticos de todo o mundo!

Open Beauty Facts é um projeto sem fins lucrativos feito inteiramente por voluntários, nós precisamos de si.

Adicionar produtos

Use nossa app Android, iPhone ou Windows Phone para digitalizar facilmente o código de barras de produtos de sua casa ou lojas locais e enviar fotos das etiquetas deles.

Sem telemóvel? Não há problema: também pode usar a sua máquina fotográfica para adicionar produtos diretamente no site.

No site também pode preencher as informações dos produtos que adicionou ou adicionados por outros.

Torná-lo local

Pode ajudar a traduzir o site e aplicação móvel para o seu idioma e para traduzir apresentações, anúncios, etc.

Podemos trabalhar juntos para oferecer suporte e adicionar logótipos para rótulos locais, para descodificar códigos do embalador local para que possamos mapear produtos, etc.

Inicie ou participe numa comunidade de colaboradores locais: adicione produtos locais, recrute amigos, apresente o projeto em encontros e conferências locais, etc.

Ajuda especializada necessária

Desenvolver o Open Beauty Facts também requer conhecimento especializado e de perito em muitas áreas diferentes:

  • Gestão de projetos

    Temos muitas ideias e você provavelmente tem ainda mais, mas é um grande desafio priorizá-las, construir um roteiro e gerir projetos quando cada participante for um voluntário com tempo limitado.

  • Design

    Precisamos de ajuda para criar uma melhor experiência do usuário no site e na aplicação móvel do Open Beauty Facts para melhorar o design, criar materiais de apresentação impactantes, etc.

  • Desenvolvimento

    Temos muito trabalho de desenvolvimento para fazer: no backend Open Beauty Facts (Perl e MongoDB), API (JSON), site (HTML5, JS, Foundation), aplicações iOS / Android / Windows Phone / FirefoxOS (desenvolvidos atualmente em HTML e JS com Apache Cordova / Phonegap), para construir novas reutilizações fiches, etc. Nosso código está em GitHub.

  • Criar comunidades

    Precisamos construir comunidades locais em todos os países e, ao mesmo tempo, uni-las globalmente.

    É muito difícil criar uma comunidade local num país sem viver lá, então sua ajuda para encontrar os primeiros participantes muito motivados é essencial. É um deles?

  • Comunicação

    Não temos certeza do que escrever aqui, você poderia ajudar?

    Mais a sério, há muitas boas coisas que os usuários, contribuidores e reusuários fazem com o Open Beauty Facts e seria ótimo para que mais pessoas soubessem disso. Em particular, precisamos de ajuda para relações públicas e de mídia.

  • Projetos Especiais

    Existem muitas aplicações interessantes e originais de dados abertos cosméticos com os quais poderíamos trabalhar com agências de cosméticos do governo, produtores de cosméticos, pesquisadores, universidades, escolas, ONGs etc. Talvez você já tenha ideias? Talvez você já tenha ideias? Se tiver, por favor nos ajude a fazé-los progredir.
