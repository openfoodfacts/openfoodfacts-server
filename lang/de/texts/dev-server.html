Die Intelligenz hinter der App und der API von Open Food Facts

Open Food Facts sammelt Daten über eine WRITE-API, verarbeitet sie und stellt sie über eine READ-API und eine Web-Version zur Verfügung. Berechnung des Nutri-Scores, NOVA-Gruppen, Herausfiltern von Zusatzstoffen, unabhängig von deren Sprache, Synonymen oder möglichen Schreibfehlern. Dies ist nur ein winziger Bruchteil der Themen, die das Serverteam abdeckt, um die Transparenz der Lebensmittel im weltumfassenden Maßstab zu gewährleisten.

Mit Perl zum Kern beitragen

Möchten Sie eine neue Sprache lernen? Der Server ist hauptsächlich in der Programmiersprache Perl programmiert. Egal, ob Sie ein erfahrener Perl-Süchtiger sind oder Perl gerade für Systemadministrationsaufgaben genutzt haben, es gibt auf Sie zugeschnittene Fehler und Betreuer, die Sie anleiten. Und auch mehr als eine Möglichkeit, es zu tun 😉

Erstellen Sie nützliche Hilfsanwendungen in einer beliebigen Sprache

HungerGames ist das Beispiel für ein kleines Spiel, das in „React” geschrieben wurde und sich positiv auf Open Food Facts auswirkt. LanguageSwitcher, eine weitere Anwendung, ist in „Python” geschrieben. Ihr Hilfsprogramm, Ihre Sprache (bitte vermeiden Sie Assembler 😉)
