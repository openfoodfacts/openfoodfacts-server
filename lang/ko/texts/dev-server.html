Open Food Facts 앱과 API에 숨겨진 지능

Open Food Facts는 WRIT API를 사용하여 데이터를 수집하고 처리하며 READ API 및 웹 버전을 사용하여 데이터를 사용할 수 있도록 합니다. Nutri-Score, 노바 그룹을 계산하고 언어, 동의어 또는 잠재적인 철자 오류에 관계없이 첨가물을 추출합니다. 이것은 지구적 규모로 음식의 투명성을 보장하기 위해 서버 팀이 다루는 주제 중 극히 일부에 불과합니다.

Perl을 사용하여 Core에 기여

새로운 언어를 배우고 싶습니까? 서버는 주로 Perl로 코딩됩니다. 여러분이 숙련된 Perl 중독자이든, 시스템 관리 업무를 위해 Perl을 다뤄본 적이 있든 상관없이 규모에 버그가 있고 지도할 멘토가 있습니다. 또한 여러가지 방법으로 기여할수 있습니다 ;-)

모든 언어로 유용한 도구를 만들기

헝거게임은 Open Food Facts에 긍정적인 영향을 미치는 반응으로 코딩된 작은 게임의 예입니다. 또 다른 도구인 LanguageSwitcher는 Python로 코딩됩니다. 당신의 도구, 언어 (어셈블러를 피하십시오 ;-)
