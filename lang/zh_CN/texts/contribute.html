需要专家帮助

开发 Open Food Facts 还需要在许多不同领域具备专业知识：

  • 项目管理

    我们有大量的想法，你可能还有更多的想法，但是要优先考虑它们，制定一个路线图，并在每个参与者通常都只是一个由有限时间的志愿者时管理项目，是一个相当大的挑战。

  • 设计

    我们需要帮助，建立一个有着更好的用户体验的 Open Food Facts 网站和移动应用，改进它们的设计，或是创建更有影响力的演示材料等。

  • 开发

    We have a lot of development work to do: on the Open Food Facts backend (Perl and MongoDB), API (JSON), web site (HTML5, JS, Foundation), iOS / Android / Windows Phone apps (a Java/Kotlin version on Android, and an hybrid version on iOS currently developed in HTML and JS with Apache Cordova / Phonegap for iOS, with a Swift version that needs volunteers), to build new cool reuses etc. We have projects in many programming languages to ensure anyone can reuse and contribute to Open Food Facts, in any language. We also have a growing Artificial Intelligence effort to simplify contribution work. 我们的代码在 GitHub 上。

  • 社区建设

    我们需要在所有国家建立当地社区，同时在全球范围内将它们联合起来。

    在一个国家中启动一个本地社区而并不生活在那里是非常困难的，因此在你的帮助下找到一个非常积极的参与者是非常重要的。 你也是其中之一吗？

  • 通讯

    我们不确定在这里写什么，你能帮忙吗？

    更重要的是，这里有许多用户、贡献者和复用者需要与 Open Food Facts 一起做的事情，让更多人了解它会是一件好事。 In particular, we need help for public and media relations.

  • 特别项目

    这里有许多关于开放数据的有趣的原生应用程序我们可以与政府机构、食品生产商、大学、非政府组织等合作。 也许你已经有想法了？ 如果你想这样做，请帮助我们推动他们前进。
