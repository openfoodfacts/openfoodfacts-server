## Format Features

1. **Parent Relationships**: Indicated by `< en: Parent Category`
2. **Multiple Synonyms**: Comma-separated for each language
3. **International Terms**: `xx:` prefix for terms used across languages
4. **Properties**: 
   - `wikidata:en: Q123456` - Wikidata Q-ID
   - `carbon_impact_fr_impactco2:en:` - CO2 equivalent
   - `unit_name:en:` - Unit name for carbon calculations

## Preserved Data from Existing Taxonomy

If a category existed in the original OPF taxonomy with additional data:
- Carbon impact values are preserved
- Custom properties are maintained
- Additional translations are merged
- Unit names for environmental calculations
