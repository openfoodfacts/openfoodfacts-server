NAME

generate_perl_html_doc_from_pod.pl scans the Perl source code of Product Opener for documentation in POD format to generate documentation in HTML files.

SYNOPSYS

The script needs to be run from the root of the Product Opener installation (e.g. /srv/off/)

    ./scripts/generate_perl_html_doc_from_pod.pl
