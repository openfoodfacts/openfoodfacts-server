Téléchargement de la table Ciqual étendue pour le calcul des apport nutritionnels CALNUT 2020

En mettant à disposition la table CALNUT, le Ciqual s'efforce de répondre au besoin des utilisateurs en fournissant une table de composition nutritionnelle sans valeurs manquantes. Attention cependant, ce fichier comporte des données issues de comblement automatique ne respectant pas les critères habituels de qualité de la table Ciqual.

Les conditions d’utilisation sont identiques à celles de la table Ciqual standard, ci-dessus.

Les données peuvent être reproduites, obligatoirement avec la référence explicite à la source :

"Anses. 2020. Table de composition nutritionnelle Ciqual pour le calcul des apport nutritionnels CALNUT"

ou, en version plus détaillée, pour une publication scientifique par exemple :

"Anses. 2020. Table de composition nutritionnelle Ciqual pour le calcul des apports nutritionnels CALNUT. Consulté le JJ/MM/AAAA. https://ciqual.anses.fr/ "
