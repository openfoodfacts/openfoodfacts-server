NAME

gen_svg_images_for_points.pl - Generates SVG images for positive or negative points for Nutri-Score components

DESCRIPTION

Each image has a number of squares corresponding to the maximum number of points for the component. The number of squares filled in green or red (for positive or negative points) corresponds to the number of points for the product. The remaining squares are filled in grey.
