→ Essayez d'être aussi précis que possible. Pour le plastique, merci d'indiquer s'il est opaque ou transparent, coloré, PET ou PEHD.

Elements d'emballage

  • Nombre d'éléments : Entrez le nombre d'éléments d'emballage de la même forme et de la même matière contenue dans le produit.
  • Forme : Entrez le nom de la forme indiqué dans les instructions de recyclage si elles sont disponibles, ou sélectionnez une forme.
  • Matière : Entrez le matériau spécifique s'il peut être déterminé (un code de matériau à l'intérieur d'un triangle peut souvent être trouvé sur les éléments d'emballage), ou un matériau générique (par exemple du plastique ou du métal) si vous n'êtes pas sûr.
  • Recyclage : Saisissez les consignes de tri seulement si elles apparaîssent sur le produit.
  • Poids d'un élément vide : Enlevez les restes de nourriture et lavez et séchez les éléments d'emballage avant de les peser. Si possible, utilisez une balance avec une précision de 0.1g ou 0.01g.
  • Quantité de produit contenue par élément : Entrez le poids net ou le volume net et indiquez l'unité (par exemple g ou ml).
